Custom CPU Execution Documentation

Class Name: CustomCPUExecution

Description: This class should provide the implementation for simulation of CPU.


public interface CPUInterface {

    void setReadyQueue(LinkedList readyQueue);

    void setWaitQueue(LinkedList waitQueue);

    void setProcess(Process process);

}

Module Details

  • This class should implement CPUInterface and extend Thread class and override the above methods and run method.


  • The method setReadyQueue sets the ready queue to be used for scheduling.


    Parameters:
    • Linked List of Process class(readyQueue)
    Return Type:
    • void

  • The method setWaitQueue sets the wait queue to be used for scheduling.


    Parameters:
    • Linked List of Process class(waitQueue)
    Return Type:
    • void

  • The method setProcess sets the process to be executed on that cpu.


    Parameters:
    • An instance of Process class
    Return Type:
    • void

  • The method run simulates the CPU execution.


    Parameters:
    • none
    Return Type:
    • void
    Details:
    • It checks whether the current burst is for CPU or IO.

    • If the current burst is for CPU, then it sleeps the thread for the specified burst time.

    • If the current burst is for IO, then it creates an IO execution thread and sends the process for IO execution.
